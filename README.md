# Radio Controlled Car

This project involves designing a wireless radio-controlled car using basic electronic circuits and signal transmission techniques.

## Description
The car is controlled remotely using RF signals, enabling forward, backward, left, and right movement.

## Components Used
- RF Transmitter & Receiver
- DC Motors
- Motor Driver (L298N)
- Battery Supply

## Working Principle
- Signals are transmitted via RF module
- Receiver processes input
- Motor driver controls movement of wheels

## Tech Stack
- Embedded Systems
- Wireless Communication
- Basic Electronics

## Future Scope
- Bluetooth/WiFi control
- Mobile app integration
- Camera-based navigation
